Welcome to the RainbowCRR technical blog.

- [Tutorials](/tutorials/)
- [About](/about/)

This site renders plain Markdown pages and Jekyll posts. Tutorials are collected from:

- standalone Markdown files in the tutorials/ directory
- blog posts in the _posts/ directory that include the category "tutorial"

To add a post that becomes a tutorial automatically, create a file in _posts/ named YYYY-MM-DD-your-title.md with front matter including "categories: [tutorial]".
