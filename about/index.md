# About RainbowCRR

This site is a minimal technical blog for RainbowCRR. Content is plain and readable. All pages are written in English.

- Tutorials are stored as separate Markdown files under `tutorials/` or as posts in `_posts/` with the category `tutorial`.
- Keep pages simple: Markdown + front matter. No heavy frameworks.
